## Introducción
La navegación permite a los usuarios moverse entre diferentes secciones de la aplicación. En esta segunda parte, nos enfocaremos en componentes avanzados para organizar la navegación: `NavigationBar`, `NavigationDrawer` y `Tabs`.

## Tabs
Las pestañas (Tabs) permiten organizar el contenido relacionado en secciones dentro de una misma pantalla. Usamos `TabRow` y `Tab`.

```kotlin
@Composable
fun TabsExample() {
    val tabs = listOf("Home", "Profile", "Settings")
    var selectedTabIndex by remember { mutableStateOf(0) }

    Column {
        TabRow(selectedTabIndex = selectedTabIndex) {
            tabs.forEachIndexed { index, title ->
                Tab(
                    selected = selectedTabIndex == index,
                    onClick = { selectedTabIndex = index },
                    text = { Text(title) }
                )
            }
        }
        
        when (selectedTabIndex) {
            0 -> HomeScreen()
            1 -> ProfileScreen()
            2 -> SettingsScreen()
        }
    }
}
```

## Navigation Bar
La barra inferior proporciona acceso rápido a las secciones principales. Debe estar dentro de un `Scaffold` y típicamente usa un `NavHost` interno para gestionar el contenido.

### Navegación Anidada (Nested Navigation)
La barra inferior no cambia de pantalla completa, sino que cambia el contenido dentro de una pantalla principal (Dashboard). Esto implica que tenemos un grafo de navegación interno.

**Estructura del Grafo:**
```text
App Navigation Graph (Principal)
├── Login
├── Register
└── Dashboard (Nested Host)
    ├── HomeUser
    ├── Search
    └── Profile
```

## Tabs vs Navigation Bar

| Característica | Tabs | Navigation Bar |
| --- | --- | --- |
| **Ubicación** | Parte superior | Parte inferior |
| **Límite de elementos** | Hasta 5-6 | Ideal 3-5 |
| **Uso principal** | Contenido relacionado en la misma pantalla | Acceso rápido a secciones principales |
| **Efecto** | Cambia contenido interno | Navega a secciones/pantallas distintas |

## Implementación de Navigation Bar

### 1. Estructura de Paquetes
Se recomienda organizar las secciones por funciones (features):
- `features/dashboard/user/UserScreen.kt`
- `features/dashboard/component/BottomNavigationBar.kt`
- `features/dashboard/navigation/UserNavigation.kt`

### 2. Componente Principal: `UserScreen`
```kotlin
@Composable
fun UserScreen(onLogout: () -> Unit) {
    val navController = rememberNavController()
    var title by remember { mutableStateOf("Inicio usuario") }

    Scaffold(
        topBar = {
            TopAppBar(title = title, logout = onLogout)
        },
        bottomBar = {
            BottomNavigationBar(
                navController = navController,
                titleTopBar = { title = it }
            )
        }
    ) { padding ->
        UserNavigation(navController = navController, padding = padding)
    }
}
```

### 3. Barra Inferior: `BottomNavigationBar`
```kotlin
@Composable
fun BottomNavigationBar(
    navController: NavHostController,
    titleTopBar: (String) -> Unit
) {
    val navBackStackEntry by navController.currentBackStackEntryAsState()
    val currentDestination = navBackStackEntry?.destination

    LaunchedEffect(currentDestination) {
        val destination = Destination.entries.find { it.route::class.qualifiedName == currentDestination?.route }
        destination?.let { titleTopBar(it.label) }
    }

    NavigationBar {
        Destination.entries.forEach { destination ->
            val isSelected = currentDestination?.route == destination.route::class.qualifiedName
            NavigationBarItem(
                label = { Text(destination.label) },
                icon = { Icon(destination.icon, contentDescription = null) },
                selected = isSelected,
                onClick = {
                    navController.navigate(destination.route) {
                        popUpTo(navController.graph.findStartDestination().id) { saveState = true }
                        launchSingleTop = true
                        restoreState = true
                    }
                }
            )
        }
    }
}

enum class Destination(val route: DashboardRoutes, val label: String, val icon: ImageVector) {
    HOME(DashboardRoutes.HomeUser, "Home", Icons.Default.Home),
    SEARCH(DashboardRoutes.Search, "Buscar", Icons.Default.Search),
    PROFILE(DashboardRoutes.Profile, "Perfil", Icons.Default.AccountCircle)
}
```

### 4. Navegación Interna: `UserNavigation`
```kotlin
@Composable
fun UserNavigation(navController: NavHostController, padding: PaddingValues) {
    NavHost(
        navController = navController,
        startDestination = DashboardRoutes.HomeUser
    ) {
        composable<DashboardRoutes.HomeUser> { UserListScreen(paddingValues = padding) }
        composable<DashboardRoutes.Search> { SearchScreen() }
        composable<DashboardRoutes.Profile> { ProfileScreen() }
    }
}
```
